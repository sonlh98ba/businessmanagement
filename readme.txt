============================================================================
Yêu cầu tối thiểu:
- Máy có cài đặt Visual Studio hỗ trợ .NET framework 4.5
- Máy có cài đặt SQL Server
============================================================================
Các bước cài đặt chương trình:
B1: Tạo database có tên TTCN2Database bằng SQL Server
B2: Import file script.sql
B3: Lấy Connection String của Database vừa import
B4: Sử dụng Visual Studio để thay đổi Connection String ở Class DataProvider nằm trong thư mục DAO
(Thư mục nằm trong chương trình)
B5: Chạy chương trình
